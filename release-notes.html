• 1.42f, 31 December 2008
    • Added the Plugins>Utilities>Capture Image command.
    • Reported UnsupportedClassVersion exceptions include the line "This plugin requires Java 1.x or later."
    • The LUT Editor (Image>Color>Edit LUT) displays the ending index in the status bar as the user drags to select a range.
    • LUT Editor color pickers no longer move the green and blue sliders when the red slider is moved.
    • Fixed a bug on Windows, introduced in v1.42e, that made it impossible to enter some characters accessed using AltGr key (e.g., "{") into macro editor windows.
    • Fixed a bug in the particle analyzer that caused it to sometimes not correctly add particles to the ROI Manager.
