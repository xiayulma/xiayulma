• 1.42c, 1 December 2008
    • Added a "Scaled Units" checkbox to the Edit>Selection>Specify dialog and a "Pixel Units" checkbox to the Analyze>Analyze Particles dialog. These checkboxes are only displayed if the image is spatially calibrated.
    • The ROI Manager "Multi Measure" command adds the ROI name to the column headers if the name does not start with a digit or if its length is less than nine characters.
    • The length of the current straight line selection is now displayed in the Analyze>Set Scale dialog with spacially calibrated images.
    • Worked around a bug on Mac OS X that required pressing the "Remove Scale" button in the Analyze>Set Scale dialog box.
